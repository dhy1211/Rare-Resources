# Phân tích hành vi khách hàng tại Jollibee 2026
## Giới Thiệu
Phân tích này ở quy mô sinh viên nên thành viên tham gia sẽ còn hạn chế 
## Công Cụ Sử Dụng 
- Ngôn Ngữ : Python
- Thư viện:Pandas (xử lý dữ liệu), Matplotlib & Seaborn (trực quan hóa).

## Các kết quả chính (Insights)
1. Món ăn yêu thích: Gà rán chiếm tỉ lệ áp đảo với 50% lựa chọn.
2. Vấn đề e ngại: 37.5% khách hàng cảm thấy phiền lòng về việc hết Voucher hấp dẫn.
3. Mức độ hài lòng: Đa số khách hàng đánh giá ở mức "Bình thường" và "Hài lòng" (chiếm hơn 60%).

## Đề xuất giải pháp
- Tối ưu hóa hệ thống quản lý Voucher trên ứng dụng để tránh tình trạng khách hàng hụt hẫng.
- Đẩy mạnh combo Gà rán kết hợp Mì ý/Kem vì đây là những món phụ được nhớ đến nhiều nhất.
